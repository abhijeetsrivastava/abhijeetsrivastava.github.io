Running plan during pandemic, 2020

January 14, 2021 3-minute read

Running

Running has been a part of my life since my bachelor’s and I have found one way or other to keep at it. During the 2020 lockdown it was difficult to run outside and I didn’t like running with my mask on. I thought of buying a treadmill to keep myself active.

So far I have made a schedule and have stuck with it for sometime now. I thought it would be a good thing to share with everyone who might find it informational / motivational.

My current schedule

Sunday Monday Tuesday Wednesday Thursday Friday Saturday
Long run Easy Run/recovery Slow run with tempo Tempo run Easy run/recovery Fast paced interval run RELAX

Long run

This comprises of running for longer interval with medium pace that varies over time. I start with warm-up, start run and then cooldown.

Easy Run/recovery

During this day, I just run for short amount of time for recovery. I accompany this with brisk walking.

Slow run with tempo

Here I run for sometime after warm-up and keep a slow pace for the entire time.

Tempo run

This run is of exact same duration as above but I pick a faster pace and keep it varying.

Fast paced interval run

I would define this as a set: 2 minutes brisk walk + run at a fast pace for 8 minutes.

I repeat this set 3-4 times.

Recovery

  1. Massaging calves helps. I do it often.
  2. Using a roller is a godsend gift. I would like to do it once a week.

One other thing that every runner does is stretching. Static stretching is really good when one would like to avoid any injury.

What I would like to achieve in 2021

My long time goal is to run a half marathon (13.1 miles or 21 km).

For this year though, I would like to run at-least 300 days. As per my plan I wont run for 52 days, and I want a cushion of 10 days at-least.

What I would recommend for beginners

Start with any c25k programs out there. For example, this is a great plan for beginners c25k hyderabad

Here are the steps I would recommend

  1. Brisk walking is another great alternative to running and one should always start with that.
  2. Either get your weight under control or get comfortable doing brisk walks.
  3. Start with a walk/run interval which should gradually translate to running.
  4. Get comfortable running for 5 minutes continuously, then 10 minutes.
  5. Register for a 5K run.

Registering for a 5K run, possibly with a buddy, is one of the best things. Do not wait for yourself to be ready. As long as you can run for 25-30 minutes, go ahead and register. Most of the 5K are chill and do not expect you to run continuously where you can do a run/walk interval. I would say, this step is a must.

Conclusion

Get started and get moving! Do not wait for a Monday or a perfect weather to start running, just get started on any day you feel like. The days you do not feel like, run anyways. This is a skill which develops overtime and there is no shortcut.

Happy Running!
